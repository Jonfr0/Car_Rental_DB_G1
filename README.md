## Car Rental Database System  

## Overview  
A **Car Rental Database System** built for the **CIS 440 Database Systems** course at **Minnesota State University, Mankato**. 

Database was normalized through **1NF, 2NF, 3NF, and BCNF**.
Manages **customers, vehicles, rentals, payments, and branch locations**.
Connected to a **PHP web app** displaying data from **MySQL**.
